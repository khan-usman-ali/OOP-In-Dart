

This is a year of service calculation calculator.In this application we have two class with name

1-Employee class
2-Main class

1-Employee class

In this employee class we have following instances name,salary and appointment year and one method with name calculateyearofservice. We will set the values using constructor of employee class. In calculate year of service we will use the datetime.now method and find the difference with the user input of yearofappointment using difference method and print it.

2- Main Class
In main class we will create a object of employee and pass values using constructor of employee. and access calculateyearofservice method using employee object.
